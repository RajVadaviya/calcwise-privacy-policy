CalcWise does not collect, store, or process any personal information from users.

Personal Information

Phone numbers

Usage Information

CalcWise does not track user behavior, usage patterns, analytics, or interaction data.

If third party services are added in future versions, this Privacy Policy will be updated accordingly.

No personal or sensitive data is transmitted over the internet.

Children’s Privacy

CalcWise does not knowingly collect information from children under the age of thirteen.

The application is suitable for users aged thirteen and above.

Data Security

Since the application does not collect or transmit personal information, there is no risk of data exposure from developer systems.
